### 五、日常維護
- 5.1 每日檢查項目

  - 後台是否可登入
  - API 是否回應正常
- 5.2 每週檢查項目
  - 訪客統計確認（Dashboard）
  - 錯誤 Log 檢查
- 5.3 每月檢查項目
  - 套件安全性更新（npm audit）
  - 資料庫備份確認
  - SSL 憑證到期日檢查

### 八、異常處理 SOP
- 8.1 前台網站無法存取
  - 排查流程：DNS → IIS → Node process → API → DB
- 8.2 後台無法登入
  - 排查流程：API 連線 → Token 過期 → 帳號狀態 → DB
- 8.3 API 回應錯誤
  - 排查流程：IIS 狀態 → Kestrel log → DB 連線 → 程式錯誤
- 8.4 資料庫連線失敗
  - 排查流程：SQL Server 服務 → 連線字串 → 防火牆 → 磁碟空間
- 8.5 頁面顯示異常（空白/排版亂）
  - 排查流程：Console 錯誤 → API 回應 → CSS 衝突 → 快取
- 8.6 部署後頁面空白 / 404
- 8.7 Token 過期 / 登入問題
